## About this project

This is a streamer-style portfolio website I built for **banks RL**.

The main idea was to make one page where people can quickly see his content, watch clips, check out his edits, and open his TikTok.

I wanted the site to look dark, clean, and a bit more polished than a basic beginner project, while still keeping the code simple enough for me to understand and improve.

---

## What it includes

* A main hero section
* TikTok/live stream area
* Clip cards
* Category filter buttons
* Popup modal for clips
* Buttons that link to TikTok
* Responsive layout for smaller screens
* Dark creator-style design

## What I practised

This project helped me practise real frontend skills instead of only doing small exercises.

Some of the main things I worked on were:

* Splitting a webpage into proper sections
* Making cards with images and text
* Styling buttons and hover effects
* Using JavaScript click events
* Using `querySelectorAll()`
* Reading data from `dataset`
* Adding and removing classes
* Opening and closing a popup
* Making the layout work better on different screen sizes

## How to run it

Clone the project:

```bash
git clone https://github.com/frvrdre/Streamer-portfolio.git
```

Go into the folder:

```bash
cd Streamer-portfolio
```

Then open `index.html` in your browser.

You can also open it with the **Live Server** extension in VS Code.
